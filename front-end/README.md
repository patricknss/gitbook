# Frontend

## Contexto

Notas práticas de frontend com foco em TypeScript, Angular e RxJS.

## Versões de referência (jun/2026)

- TypeScript: **6.0.3**
- Angular: **22.0.x**
- RxJS: **7.8.2**

## O que está documentado

1. Fundamentos de TypeScript (tipagem, interfaces, classes).
2. Estrutura básica de projetos Angular.
3. Componentes, templates e data binding.
4. Serviços e consumo de APIs.
5. Noções iniciais de RxJS.

## Exemplo prático

Projeto: **Tela de listagem de produtos**.

1. Criar tipos e interfaces em TypeScript.
2. Montar componente Angular para lista e filtro.
3. Consumir API via serviço com `HttpClient`.
4. Usar RxJS para busca reativa com debounce.
