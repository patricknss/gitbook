# Trilha de Backend

Esta trilha foi reorganizada para focar em aprendizagem progressiva com critérios objetivos de sucesso.

## Versões de referência (jun/2026)

- .NET: **10.0 (LTS)** (SDK 10.0.301)
- C#: **14**
- ASP.NET Core: **10**

## Critério de sucesso

1. Fundamentos de C# (tipos, coleções, exceções).
2. Programação orientada a objetos (classes, herança, interfaces).
3. Conceitos básicos de ASP.NET (controllers, services).
4. Async/await e noções básicas de concorrência.

## Ordem sugerida

1. Fundamentos de C#
2. Programação Orientada a Objetos
3. ASP.NET básico
4. Async/await e concorrência
