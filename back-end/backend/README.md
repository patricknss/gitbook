# Backend (Fundamentos e APIs)

Conteudo pratico de backend com foco em .NET/C# e fundamentos de APIs.

## Versoes de referencia (jun/2026)

- .NET: **10.0 (LTS)** (SDK 10.0.301)
- C#: **14**
- ASP.NET Core: **10**

## Topicos

1. Fundamentos de C#.
2. Programacao orientada a objetos.
3. Colecoes, arrays e LINQ.
4. ASP.NET basico (controllers e services).
5. Async/await e concorrencia.
6. Cache em memoria com `IMemoryCache`.

As bibliotecas foram centralizadas na pasta raiz [Bibliotecas](../../bibliotecas/README.md).
