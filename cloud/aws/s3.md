# Amazon S3

Amazon S3 é armazenamento de objetos altamente durável.

## Casos comuns

- Upload de arquivos de usuários.
- Armazenamento de imagens, PDFs e backups.
- Hospedagem de conteúdo estático.

## Conceitos

- **Bucket**: contêiner lógico dos objetos.
- **Object**: arquivo + metadados.
- **Policy**: regra de acesso ao bucket.

## Exemplo de upload seguro

1. Backend gera URL pré-assinada (presigned URL).
2. Frontend faz `PUT` direto no S3 com essa URL.
3. Backend salva no banco apenas a chave do arquivo (`object key`).
