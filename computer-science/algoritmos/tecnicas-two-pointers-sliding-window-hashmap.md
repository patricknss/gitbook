# Tecnicas: Two Pointers, Sliding Window e Hash Map

Padroes para resolver problemas de array/string com eficiencia.

## Two pointers

Usa dois indices para reduzir comparacoes.

Exemplos:
- par com soma alvo;
- remover duplicados em array ordenado.

## Sliding window

Mantem uma "janela" valida e move inicio/fim.

Exemplos:
- maior substring sem repeticao;
- soma maxima de subarray de tamanho `k`.

## Hash map / hash set

Usa busca media `O(1)` para frequencia e existencia.

Exemplos:
- two sum;
- contagem de ocorrencias.
