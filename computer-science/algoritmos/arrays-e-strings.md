# Arrays e Strings

Base da maioria dos problemas de entrevista e do dia a dia.

## Operacoes comuns

- acesso por indice;
- busca linear;
- insercao/remocao com deslocamento;
- inversao e rotacao.

## Padroes importantes

1. Two pointers.
2. Sliding window.
3. Prefix sum.
4. Hash map para lookup rapido.

## Erros comuns

- esquecer limite de indice;
- copiar array inteiro sem necessidade;
- usar concatenacao repetida de string em loop.
