# Busca e Ordenacao

Algoritmos base para localizar e organizar dados.

## Busca

- busca linear: `O(n)`;
- busca binaria (dados ordenados): `O(log n)`.

## Ordenacao (visao geral)

| Algoritmo | Complexidade media |
| --- | --- |
| Bubble/Insertion | `O(n^2)` |
| Merge sort | `O(n log n)` |
| Quick sort | `O(n log n)` medio |
| Heap sort | `O(n log n)` |

## Regra pratica

- para colecoes pequenas, simplicidade pode bastar;
- para volume alto, priorize `O(n log n)`.
