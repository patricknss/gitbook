# Arvores, BST e Trie

Estruturas hierarquicas para busca e organizacao.

## Arvore binaria

Cada no tem ate dois filhos.

## BST (Binary Search Tree)

- esquerda < no < direita;
- busca media `O(log n)` em arvore balanceada.

## Trie

- arvore de prefixos para strings;
- excelente para autocomplete e busca por prefixo.

## Travessias

- pre-order, in-order, post-order, level-order.
